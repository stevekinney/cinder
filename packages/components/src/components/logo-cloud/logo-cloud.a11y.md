# LogoCloud · accessibility

## Pattern

LogoCloud presents partner/customer logos as supporting social proof.

## Keyboard and focus

- Linked logos must remain keyboard focusable.
- Non-linked logos should not appear in tab order.

## Names, roles, and state

- Provide a meaningful `name` for each logo so image `alt` text is useful.
- Keep grayscale styling purely visual; it must not indicate state.

## Verification

- Tab through linked logos and ensure each has a visible focus indicator.
- Inspect logo image names in accessibility tools to confirm usable alt text.

Related components: `team-section`, `testimonial-section`, `blog-section`.
